# GestureWave - Распознавание жестов с визуальной обратной связью

## Описание проекта

**GestureWave** - это интерактивное веб-приложение для распознавания жестов рук в реальном времени с использованием веб-камеры. Система определяет положение рук, распознает более 10 различных жестов и выводит соответствующее забавное изображение (мем) в качестве визуальной обратной связи.

Проект разработан в рамках курсовой работы по дисциплине "Создание программного обеспечения" и демонстрирует интеграцию технологий компьютерного зрения и веб-разработки.

### Основные возможности

- Захват видеопотока с веб-камеры в реальном времени
- Детекция рук с выделением 21 ключевой точки на каждой кисти
- Распознавание 10+ жестов (Мир, Открытая ладонь, Кулак, Рок, Вишенка на торте и другие)
- Визуальная обратная связь - вывод названия жеста и соответствующего мема
- Поддержка двух рук одновременно
- Отображение служебной информации (FPS, количество рук)
- Автосохранение координат ключевых точек в JSON-формат (каждые 3 секунды)
- Регистрация и авторизация пользователей
- Анимированный 3D-фон с эффектом звёздного неба (Three.js)
- Интуитивно понятный интерфейс без сложных настроек

## Технологический стек

### Бэкенд

- **Python 3.8+** - основной язык программирования
- **Flask** - веб-фреймворк для создания REST API
- **OpenCV** - захват и обработка видеопотока
- **MediaPipe** - детекция рук и распознавание жестов
- **SQLite** - лёгковесная база данных для хранения пользователей и сессий

### Фронтенд

- **HTML5 / CSS3** - структура и стилизация интерфейса
- **JavaScript (ES6)** - клиентская логика
- **Three.js** - создание анимированного 3D-фона
- **Fetch API** - асинхронное взаимодействие с сервером

## Установка и запуск

### Системные требования

- Python 3.8 или выше
- Веб-камера
- Операционная система: Windows / Linux / macOS

### Пошаговая инструкция

#### 1. Клонирование репозитория

```bash
git clone https://github.com/ISHIKHIN/Gesture.git
cd Gesture
```
#### 2. Создание и активация виртуального окружения
#### Windows:
```bash
python -m venv venv
venv\Scripts\activate
```
#### Linux / macOS:
```bash
python3 -m venv venv
source venv/bin/activate
```
#### 3. Установка зависимостей
```bash
pip install --upgrade pip
pip install -r requirements.txt
```
#### 4. Создание необходимых директорий
```bash
mkdir instance static\images data logs
```
#### 5. Запуск приложения
```bash
python app.py
```
После запуска приложение будет доступно по адресу: http://127.0.0.1:5000

### Повторный запуск:
#### Windows:
```bash
cd Gesture
venv\Scripts\activate
python app.py
```
#### Linux / macOS:
```bash
cd Gesture
source venv/bin/activate
python app.py
```

### Описание ключевых файлов и модулей

| Файл | Назначение |
|------|------------|
| `app.py` | Основной файл приложения. Содержит Flask-приложение, все API-эндпоинты, поток захвата видео и генерацию MJPEG-потока. |
| `auth.py` | Blueprint для маршрутов аутентификации. Содержит декоратор `login_required` для защиты маршрутов. |
| `database.py` | Инициализация базы данных SQLite. Содержит модели пользователей и сессий, функции хеширования паролей и операции работы с БД. |
| `hand_tracker.py` | Класс HandTracker. Инициализирует MediaPipe, обрабатывает кадры и распознаёт жесты. |
| `requirements.txt` | Список всех зависимостей проекта с указанием версий. |
| `templates/` | HTML-шаблоны для рендеринга страниц. |
| `static/` | Статические файлы: CSS-стили, изображения-мемы, клиентские JavaScript-скрипты. |
| `data/` | Директория для сохранения JSON-файлов с координатами ключевых точек рук. |
| `logs/` | Директория для хранения логов работы приложения. |
| `instance/` | Директория для хранения файла базы данных SQLite (`app.db`). |
| `venv/` | Виртуальное окружение Python со всеми установленными зависимостями. |
